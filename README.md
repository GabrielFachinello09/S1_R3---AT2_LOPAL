# Exercício 1- Número Ímpar e Par (Exerc-1.py)

O código tem como objetivo percorrer todos os números de 0 até 100 e identificar se cada valor é par ou ímpar. Para isso, foi utilizada a estrutura de repetição for junto da função range(0,101), responsável por gerar a sequência numérica. Durante cada repetição, o programa aplica uma verificação utilizando o operador módulo %, que retorna o resto da divisão do número por 2. Quando o resultado da operação é igual a 0, o número é classificado como par, caso contrário, é considerado ímpar. Em seguida, o programa exibe uma mensagem formatada informando o número analisado e sua respectiva classificação.

# Exercício 2- Identificação de Maior e Menor da Lista (Exerc-2.py)

O código tem como objetivo receber três números digitados pelo usuário e identificar qual deles é o menor e qual é o maior valor informado. Para isso, o programa utiliza a função input() para capturar os números inseridos pelo usuário e a função int() para converter os valores para o tipo inteiro. Em seguida, os números são armazenados em uma lista, permitindo que sejam manipulados de forma organizada. Após isso, é utilizado o método sort(), responsável por ordenar os elementos da lista em ordem crescente. Com a lista organizada, o programa acessa o primeiro elemento da lista, que representa o menor número, e o último elemento, que representa o maior número informado. Por fim, o código exibe uma mensagem formatada apresentando os resultados encontrados.

# Exercício 3- Nome Decomposto   Verticalmente (Exerc-3.py)

O código tem como objetivo receber um nome digitado pelo usuário e exibi-lo de forma progressiva na vertical, mostrando uma letra adicional a cada linha. Para isso, o programa utiliza a função input() para capturar o nome informado pelo usuário e a função len() para calcular a quantidade de caracteres presentes na palavra digitada. Em seguida, é utilizada a estrutura de repetição for junto da função range(), responsável por percorrer os índices do nome do primeiro até o último caractere. Durante cada repetição, o programa utiliza o fatiamento de strings (nome[:i]) para exibir apenas uma parte do texto, aumentando gradativamente a quantidade de letras mostradas na tela. Dessa forma, o nome é apresentado de maneira crescente até ser exibido completamente. Este exemplo demonstra conceitos importantes da programação em Python, como manipulação de strings, estruturas de repetição, contagem de caracteres e fatiamento de texto.

# Exercício 4- Sequência Fibonacci (Exerc-4.py)

O código tem como objetivo gerar e exibir os termos da sequência de Fibonacci de acordo com a quantidade informada pelo usuário. Inicialmente, o programa utiliza a função input() para solicitar ao usuário o número de termos desejados e a função int() para converter o valor digitado em um número inteiro. Em seguida, são definidas as variáveis penultimo e ultimo, ambas iniciadas com o valor 1, representando os primeiros termos da sequência de Fibonacci. O código também utiliza estruturas condicionais (if, elif) para tratar situações específicas, como valores menores ou iguais a zero e casos em que o usuário deseja apenas um ou dois termos da sequência. Após essas verificações, é utilizada a estrutura de repetição while, responsável por calcular os próximos números da sequência até atingir a quantidade solicitada. Durante cada repetição, o programa soma os dois últimos valores para gerar o próximo termo da sequência, atualizando as variáveis e exibindo os resultados na tela.

# Exercício 5- Questionário Pessoal (Exerc-5.py)

O código tem como objetivo realizar um questionário interativo para coleta e validação de informações pessoais de um usuário, simulando um sistema de cadastro empresarial. Inicialmente, o programa exibe uma mensagem de boas-vindas e solicita o primeiro nome do usuário, utilizando a função input(). Em seguida, a função len() é aplicada para verificar a quantidade de caracteres do nome informado, permitindo que o sistema identifique se o nome possui mais, menos ou exatamente três letras. Após essa etapa, o código utiliza estruturas de repetição while para garantir que determinadas informações sejam preenchidas corretamente antes do prosseguimento do cadastro. Durante o preenchimento da idade, o sistema valida se o valor informado é coerente e, posteriormente, classifica o usuário em diferentes faixas etárias, como criança, adolescente, jovem adulto, pessoa de meia-idade ou terceira idade. Na etapa salarial, o programa compara o valor digitado com o salário mínimo nacional e apresenta mensagens específicas de acordo com a situação financeira informada, incluindo alertas relacionados à informalidade no trabalho. O sistema também realiza validações para os campos de sexo e estado civil, aceitando apenas respostas específicas previamente definidas, garantindo maior consistência nas informações armazenadas. Ao final do processo, todas as respostas fornecidas são exibidas para conferência, permitindo que o usuário confirme se os dados estão corretos.
